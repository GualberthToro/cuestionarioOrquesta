Cuestionario

Diseño de un sistema de gestión administrativa para la orquesta.

El presente instrumento tiene como propósito recopilar información que permita analizar el contexto organizativo actual y diagnosticar la necesidad de automatizar los procesos administrativos de la orquesta. Su finalidad es apoyar el desarrollo de una aplicación de escritorio como solución tecnológica. Este cuestionario es anónimo, y su participación es voluntaria, garantizando la confidencialidad de las respuestas.

Instrucciones:

Lea cuidadosamente cada pregunta y marque con una "X" la respuesta que más se ajuste a su opinión o conocimiento.

Sección I: Procesos administrativos actuales

  1. ¿Considera usted que los procesos administrativos actuales son eficientes?
  2. ¿El registro de la información se realiza de manera organizada?
  3. ¿Tiene usted acceso rápido a la información cuando la necesita?
  4. ¿Los documentos administrativos se almacenan adecuadamente?
  5. ¿Se presentan errores con frecuencia en el manejo manual de la información?

Sección II: Uso y disposición hacia herramientas tecnológicas

  1. ¿Está usted familiarizado(a) con el uso básico de computadoras?
  2. ¿Considera necesaria la implementación de un sistema automatizado?
  3. ¿Estaría de acuerdo en digitalizar los registros?
  4. ¿Cree usted que el sistema informático facilitaría su trabajo?
  5. ¿Estaría usted dispuesto(a) a aprender a usar un nuevo sistema?

Sección III: Expectativas del nuevo sistema

  1. ¿Considera que el nuevo sistema debe permitir registrar información claramente?
  2. ¿Debería incluir funciones para modificar y consultar datos?
  3. ¿Sería útil que el sistema genere reportes automáticamente?
  4. ¿El sistema debería ser fácil de usar para todo el personal?
  5. ¿Es importante que el sistema tenga copias de seguridad?
